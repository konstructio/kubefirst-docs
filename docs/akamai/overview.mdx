Akamai is in beta. Use at your own risks. The Akamai documentation is missing proper diagrams & some images, and will display Civo ones, in the meantime: it will be fixed soon.

:::

[Linode has been acquired by Akamai in 2022](https://www.akamai.com/newsroom/press-release/akamai-completes-acquisition-of-linode), but the branding is still in some cases, like for the CLI used to connect to Kubernetes on Akamai. If you see Linode written in the documention, keep in mind that it is in reality Akamai, but that we have to use the previous brand because Akamai still do.

## Known Limitations

<KnownLimitations />

### Akamai Specific

- Akamai is [only available on GitHub](https://github.com/kubefirst/kubefirst/issues/2143) for now.
- You [cannot use Akamai as the DNS provider](https://github.com/kubefirst/kubefirst/issues/2144) for now: they have a requirement with domain that needs to be attached to a VM that we need to sort out. For now, you can use Cloudflare as the DNS provider without any issues.
