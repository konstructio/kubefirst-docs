## Known Limitations

<KnownLimitations />

### K3s Specific

- Cloudflare is the only DNS provider implemented so far.
- K3s [cannot be installed using the UI console](https://github.com/kubefirst/kubefirst/issues/2147): you can only use the CLI for now.
- K3s is [only available on GitLab](https://github.com/kubefirst/kubefirst/issues/2146) for now.
