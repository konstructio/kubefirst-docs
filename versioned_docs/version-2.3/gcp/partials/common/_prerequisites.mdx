### Google Cloud Prerequisites

For kubefirst to be able to provision your Google Cloud resources:

- A [Google Cloud account](https://cloud.google.com) in which you are an account owner.
- A publicly routable [DNS](https://cloud.google.com/dns/docs/overview).
- The `GOOGLE_APPLICATION_CREDENTIALS` environment variable set with the credential JSON file (see [Google Cloud documentation](https://cloud.google.com/docs/authentication/application-default-credentials) for more information on how to generate it).
